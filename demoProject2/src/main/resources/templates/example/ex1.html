타임리프 예제 1

			Spring EL(스프링 표현 언어)
			- ${key} : 변수, Model 등을 이용해서 세팅한 값 출력
			
			th:text 속성 = "속성값"
			- 타임리프 이용해서 속성 값을 작성된 태그의 내용(innerText)으로 출력
			
			** th 속성은 출력된 화면(응답화면) 보여지지 않음 **
			--> 왜냐하면 해석된 후 사라짐

단일 값 출력

			th : block 태그
			- Thymeleaf 에서 제공하는 유일한 태그
			
			- Thymeleaf 는 태그 내에 th 속성을 작성하여 저으이하는게 일반적이지만
			  th 속성을 작성할 태그가 존재하지 않을 때 사용
			  
			- th : text 보단 조건, 반복문과 같이 사용하는 경우가 많음
productName : 상품명 / 상품명 price : 가격 원 productCompany : 회사명

복수 값 출력

			th:each="item : ${List 또는 배열}"
			
			- 향상된 for 문
			
			- List 또는 배열 길이 만큼 반복
			
			- 매 반복시마다 List 또는 배열의 요소를 차례대로 꺼내 item 변수에 저장
				(item 변수명은 자유롭게 잓성)
